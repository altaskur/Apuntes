# Guía de estilos sobre un bloque de css

Esta guía se basa en la estructura sugerida por Airbnb y estructuramos
los bloques de la siguiente manera:

```css
    section{
        /* BoxModel */
        display:flex;
        width:auto;
        height:100%;
        margin: 0;
        position:top;

        /* Tipografía y texto */
        font-family:Arial;
        font-size:bold;
        font-weight:18pt;
        color: blue;
        position:top;

        /* Apariencia visual */
        background-color:Arial;
        border:bold;
        box-shadow:18pt;
        opacity: blue;

        /* Otras propiedades */
        overflow:none;
        cursor:pointer;
        outline:none;
        transform: 1s all;
    }
```
